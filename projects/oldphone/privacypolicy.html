OldPhone does not collect nor record any user data. It does not share any of the user data it doesn't collect. The data it doesn't collect isn't stored on the device, and it doesn't transmit the non-existent data to any server.

It connects to the network in order to provide its services. These services include:

  • Camera
  • Doorbell
  • Motion detector
  • Remote control
  • File store
  • Accelerometer
  • GPS
  • Wifi
  • Bluetooth
  • Audio

It will access your files, in order to share them with you, over the network.
