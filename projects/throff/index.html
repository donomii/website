Throff

Throff is a dynamically typed, late binding, homoiconic, concatenative programming language.

Throff is a new programming language, designed to make my work at the command line easier, and to explore the design of programming languages. There is extensive documentation, and a growing library of examples

The documentation contains beginners guide and command reference, and there are articles explaining some of the design and internal workings.

Throff's goal is to be useful at the command line, and for writing larger programs.
