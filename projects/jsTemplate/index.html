jsTemplates let you write web pages in markdown, and render them on the client's web browser. This gives you the clean, simple style of markdown, and the power of html.

jsTemplates works without any server configuration, allowing you to deploy your webpages on any host service. All the rendering occurs on the user's computer, saving you server costs. You don't need server side scripting languages like PHP, Perl or Ruby. Your webpage can be viewed on any machine, without a web server.

jsTemplates gracefully degrades to plain html, if the client does not support javascript. Unfortunately, search engines do not extract URLs from markdown text, so any website using this library is invisible to all search engines. Which might be a plus, in this age of AI scrapers.
